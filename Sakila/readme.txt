To make use of the program ensure that you have the sakila database in your
mysql library

-Input your mysql database credentials in the config.txt, don't put any spaces

-Ensure the mariadb-java-client-3.0.4.jar file is included in the classPath, if you don't have the jar file,
 then find it in the extra folder

-Open the project and run the App.java file

-If you don't have the sakila database, use the resources in the extra folder
